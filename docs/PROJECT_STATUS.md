# Estado del proyecto

La versión 1.0.0 establece la plataforma evolutiva. No representa una finalización científica.

| Área | Estado | Próximo objetivo |
|---|---|---|
| Secuencias y traducción | validated | códigos genéticos adicionales y property tests |
| Variantes sobre CDS | validated | anotaciones por transcrito y normalización |
| FASTA/GFF3/VCF | validated, alcance inicial | escritura, compresión e índices |
| Perfiles de organismo | validated como contrato | perfiles taxonómicos específicos |
| G-cuádruplex | experimental | benchmarks y métodos alternativos |
| Mosaicismo | experimental | modelos de error y calidad |
| Pangenoma | concept | contrato de coordenadas en grafo |
| Regulación y epigenética | roadmap | modelos BED/bedGraph y pistas |
| Célula única y espacial | roadmap | formatos y matrices dispersas |

La fuente ejecutable del estado es `registry/modules.json`; esta tabla es una lectura humana complementaria.
