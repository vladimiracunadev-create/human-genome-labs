# Perfiles de organismo

Los algoritmos no deben asumir automáticamente genoma humano nuclear, doble hebra lineal o código estándar. `packages/organism-profiles/data/profiles.json` separa:

- organismo y taxonomía;
- compartimento;
- ADN o ARN;
- una, dos o configuración variable de hebras;
- topología;
- segmentación;
- código genético;
- ensamblaje predeterminado;
- limitaciones.

Los perfiles genéricos son conceptos que deben especializarse antes de un análisis real.
