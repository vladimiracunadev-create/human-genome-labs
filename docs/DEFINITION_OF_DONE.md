# Definición de terminado

Una capacidad solo puede declararse validada cuando cumple:

- [ ] contrato y tipos;
- [ ] validación de entradas;
- [ ] pruebas unitarias;
- [ ] prueba con datos conocidos;
- [ ] documentación científica;
- [ ] fuentes trazables;
- [ ] limitaciones explícitas;
- [ ] ejemplo CLI o de biblioteca;
- [ ] salida reproducible;
- [ ] manifiesto y registro actualizados;
- [ ] CHANGELOG actualizado.

Las capacidades experimentales pueden estar incompletas, pero nunca deben ocultar esa condición.
