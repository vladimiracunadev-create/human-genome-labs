# Human Genome Labs 1.0.0

Primer release estable. Establece una plataforma científica evolutiva, reproducible y educativa para genética molecular y genómica, con un núcleo verificable y un mecanismo para incorporar investigación mediante módulos con madurez, evidencia, versiones y limitaciones explícitas.

Incluye el núcleo de análisis de secuencias y variantes, contratos `ScientificResult<T>`, perfiles de organismo, lectores FASTA/GFF3/VCF, base científica por dominios, registro de módulos, CLI, laboratorio web, juego secundario y distribución web precompilada.

No significa que el campo esté cerrado: la versión 1.0 define cómo crecer sin mezclar ciencia, interfaz y experimentación. Consulte el [roadmap](docs/ROADMAP.md).
